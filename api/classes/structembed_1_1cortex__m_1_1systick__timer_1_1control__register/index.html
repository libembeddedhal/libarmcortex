        Detailed Description

        struct embed::cortex_m::systick_timer::control_register;

        Namespace containing the bitmask objects that are used to manipulate the ARM Cortex Mx SysTick Timer.

        Public Attributes Documentation

        variable enable_counter

        static constexpr auto enable_counter = xstd::bitrange::from<0>();

        When set to 1, takes the contents of the reload counter, writes it to the current_value register and begins counting down to zero. Setting this to zero stops the counter. Restarting the counter will restart the count.

        variable enable_interrupt

        static constexpr auto enable_interrupt = xstd::bitrange::from<1>();

        When SysTick timer’s count goes from 1 to 0, if this bit is set, the SysTick interrupt will fire.

        variable clock_source

        static constexpr auto clock_source = xstd::bitrange::from<2>();

        If set to 0, clock source is external, if set to 1, clock source follows the processor clock.

        variable count_flag

        static constexpr auto count_flag = xstd::bitrange::from<16>();

        Set to 1 when count falls from 1 to 0. This bit is cleared on the next read of this register.
